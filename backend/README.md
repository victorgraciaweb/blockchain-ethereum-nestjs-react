# Ejecutar en desarrollo

1. Clonar el repositorio
2. Ejecutar
```
yarn install
```
3. Tener Nest CLI instalado (Opcional)
```
npm i -g @nestjs/cli
```

7. Ejecutar la aplicación en dev:
```
yarn start:dev
```
8. Importar colección con Endpoints en Postman (Opcional) 
- `blockchain.postman_collection.json`

## Stack usado
* Nest
